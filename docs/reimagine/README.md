## Quick Reference

### Supported Formats

- JPEG, PNG, WebP
- Max size: 10MB
- Recommended: 1920x1080 or higher

### Processing Times

- Enhancement: 15-30s
- Virtual Staging: 30-60s
- Day-to-Dusk: 30-45s
- Item Removal: 45-90s
- Renovation: 60-90s

### Common Workflows

**Empty Property:**

1. Upload → 2. Enhancement → 3. Virtual Staging → 4. Download

**Exterior Shot:**

1. Upload → 2. Enhancement → 3. Day-to-Dusk → 4. Download

**Cluttered Interior:**

1. Upload → 2. Enhancement → 3. Item Removal → 4. Download

**Fixer-Upper:**

1. Upload → 2. Enhancement → 3. Renovation → 4. Download
