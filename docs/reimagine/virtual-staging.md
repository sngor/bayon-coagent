# Virtual Staging Guide

Transform empty rooms into beautifully furnished spaces that help buyers visualize living in the property.

## What is Virtual Staging?

Virtual staging uses AI to add realistic furniture, decor, and accessories to empty room photos. It's a cost-effective alternative to physical staging that can be done in minutes.

## When to Use Virtual Staging

‚úÖ **Perfect for:**

- Vacant properties with empty rooms
- New construction before furniture arrives
- Properties where physical staging is too expensive
- Rooms that need to show multiple layout options

‚ùå **Not ideal for:**

- Already furnished rooms (use renovation instead)
- Outdoor spaces (no furniture needed)
- Very dark or poorly lit photos

## Before You Start

### Image Requirements

- **Room should be empty** or mostly empty
- **Good lighting** - natural light preferred
- **Clear view** of the entire room
- **Straight angles** - avoid extreme perspectives
- **Clean space** - remove clutter first

### Best Photo Practices

1. **Shoot from doorway height** (about 5 feet)
2. **Include full room** in frame
3. **Turn on all lights** for even lighting
4. **Open curtains/blinds** for natural light
5. **Clean floors and walls** before shooting

## Using Virtual Staging

### Step 1: Select Room Type

Choose the room type that matches your photo:

- **Living Room**: Main gathering spaces, family rooms
- **Bedroom**: Master, guest, or children's bedrooms
- **Kitchen**: Dining areas within kitchens
- **Dining Room**: Formal or casual dining spaces
- **Office**: Home offices, studies, dens
- **Bathroom**: Full or half bathrooms

üí° **Tip**: Selecting the correct room type ensures appropriate furniture placement.

### Step 2: Choose Furniture Style

Select a style that matches the property's aesthetic:

#### Modern

- Clean lines and minimalist design
- Neutral colors with bold accents
- Contemporary furniture pieces
- **Best for**: Urban condos, new construction

#### Traditional

- Classic furniture with ornate details
- Warm, rich colors
- Timeless pieces
- **Best for**: Historic homes, suburban houses

#### Minimalist

- Simple, functional furniture
- Monochromatic color schemes
- Uncluttered spaces
- **Best for**: Small spaces, modern apartments

#### Luxury

- High-end furniture and finishes
- Rich textures and materials
- Statement pieces
- **Best for**: High-value properties, penthouses

#### Rustic

- Natural wood and materials
- Earthy color palettes
- Cozy, warm atmosphere
- **Best for**: Cabins, farmhouses, country homes

#### Contemporary

- Mix of modern and traditional
- Comfortable and stylish
- Versatile design
- **Best for**: Most residential properties

### Step 3: Review and Adjust

After AI generates the staging:

1. **Check furniture scale** - Does it fit the room naturally?
2. **Verify style consistency** - Does it match the property?
3. **Look for placement issues** - Is furniture in logical positions?

If results aren't perfect:

- Click **Regenerate** to try again
- Try a different style
- Ensure room type is correct

## Best Practices

### Do's ‚úÖ

- **Match the property's price point** - Luxury staging for luxury homes
- **Consider the target buyer** - Family furniture for family homes
- **Keep it realistic** - Don't over-furnish small spaces
- **Maintain consistency** - Use similar styles across all rooms
- **Show functionality** - Help buyers understand room purpose

### Don'ts ‚ùå

- **Don't over-stage** - Leave some breathing room
- **Avoid trendy styles** - Stick to timeless looks
- **Don't mismatch styles** - Keep aesthetic consistent
- **Avoid blocking features** - Don't hide fireplaces, windows, etc.
- **Don't stage outdoor spaces** - Use other edit types instead

## Common Issues and Solutions

### Issue: Furniture looks too small or too large

**Cause**: Room dimensions not properly detected
**Solution**:

- Ensure photo shows full room
- Retake photo from standard height
- Try regenerating with different parameters

### Issue: Furniture placement seems odd

**Cause**: Room layout not clearly visible
**Solution**:

- Make sure floor is clearly visible
- Remove any existing furniture first
- Ensure good lighting throughout room

### Issue: Style doesn't match property

**Cause**: Wrong style selected
**Solution**:

- Review style descriptions carefully
- Consider property age and location
- Try multiple styles to compare

### Issue: Colors clash with room

**Cause**: AI didn't detect wall/floor colors accurately
**Solution**:

- Improve photo lighting
- Ensure white balance is correct
- Try regenerating for different color options
