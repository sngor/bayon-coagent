# **App Name**: AuthorityOS for Real Estate

## Core Features:

- Unified Profile Management: Centralize agent profile data (name, licenses, experience, bio) to create a single source of truth for expertise. This information can then be leveraged across all connected platforms, ensuring brand consistency and saving agents valuable time.
- Automated Schema Generation: Automatically generate RealEstateAgent and RealEstateAgency schema markup in JSON-LD format using profile data, making it easier for search engines to understand agent identity and services.
- Reputation and Review Hub: Aggregate client reviews from Google, Zillow, and Yelp into a central dashboard for reputation management, monitor NAP consistency to strengthen local SEO signals.
- AI-Assisted Content Creation: Leverage generative AI tool to draft hyper-local content such as neighborhood guides, using public data and market statistics to create comprehensive resources, then the agent add their personal insights.
- Seamless IDX Integration: Embed live, filtered IDX feeds showing properties for sale within specific neighborhood guides, converting informational searches into active leads and providing users with an up-to-date and engaging experience.
- Persona-Driven Listing Descriptions: Automatically rewrite property descriptions to target specific buyer personas (e.g., First-Time Homebuyer, Luxury Investor), highlighting features most important to each persona using conversational language.
- Automated Listing Schema: Generate RealEstateListing, Offer, and Event schema markup for each listing to provide clear, structured signals to search engines, ensuring optimal visibility in AI-driven search results.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB). This calming color inspires trust and stability. Blue is frequently used in the real estate sector because of the feeling of trust it inspires. In HSL it will have the same hue as the background color, but with higher saturation and lower luminance.
- Background color: Light grayish-blue (#E6F3FF). Light background, as typical for data-intensive interfaces, that lets the content stand out. This background color will also set off the app's informative and professional nature.
- Accent color: Muted teal (#70A4A9). The teal provides a calm sense of forward direction, being as it is related to blue.
- Body font: 'PT Sans', sans-serif, for body text.
- Headline font: 'Playfair', serif, for headlines.
- Simple, clean icons representing various real estate features and actions.
- Clean and intuitive layout with clear sections for each module.
